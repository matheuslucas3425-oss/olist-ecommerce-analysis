# olist-ecommerce-analysis
Simple Data Analysis Project using SQL and the Brazilian Olist dataset

E-commerce Sales Analysis – Olist Dataset

Project Overview

This project is a simple data analysis using the Brazilian Olist public dataset.  
The goal was to explore sales data and extract basic business insights using SQL.


Business Questions

The analysis answers the following questions:

1. How many customers exist in each state?
2. How many orders were placed per month?
3. What is the total revenue by state?
4. Which product categories sell the most?
5. 

Tools Used

- PostgreSQL
- SQL
- Power BI (optional)
- Kaggle Olist Dataset
- 

Dataset

The dataset used is the public Brazilian E-commerce dataset provided by Olist on Kaggle.


Key Insights

- Most customers are concentrated in the state of São Paulo.
- Orders show monthly seasonality.
- Revenue is highly concentrated in a few states.
- Some product categories dominate total sales.

Files in this Repository

- `queries.sql` – SQL queries used in the analysis
